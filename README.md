# Trivia Challenge

**Trivia Challenge** is a fun and responsive web-based quiz game built with HTML, CSS, and JavaScript. It features over **500 trivia questions** spanning decades from the 1920s to the 2020s, organized by topic and decade.

---

## 🧠 Features

- 🎯 Multiple topics: Science, History, Pop Culture, Sports, Technology  
- 🕹️ Interactive quiz interface with instant feedback  
- ⏱️ Countdown timer per question with visual + sound cues  
- 🔊 Sound effects for correct/wrong answers and ticking clock  
- 🔇 Mute/unmute toggle with saved preference  
- 🎵 Background music with auto-pause during gameplay  
- 🏆 Final score display with emoji feedback and confetti  
- 🧠 High score tracking via localStorage  
- ⌨️ A/B/C/D keyboard support for fast answering  
- 📱 Fully responsive on desktop and mobile
